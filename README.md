# Ropable

Ropeable은 2d 로프 액션 플랫포머 게임입니다. 스시를 좋아하는 작은 영혼과 함께 길을 떠나 보세요!
[플레이](https://dancydangames.itch.io/ropable)

## 게임 내용
- 10개의 레벨
- 각 레벨마다 놓여 있는 3개의 스시를 먹으며 레벨 클리어
- 온라인 리더보드로 클리어 시간 경쟁 가능

## 여담
